# EnergyLens Freelance — Service Definitions (SOP)

> **Purpose:** lock each offer so (a) the customer knows exactly what they get before they arrive,
> and (b) you do the **same repeatable steps every time** — no bespoke work, no scope creep, low
> fatigue. The sample report is the showcase; this is the operating spec behind it.
> **Pricing = reference-phase (discounted)** to win the first 2–3 references; raise once you have reviews.

---

## Offer 1 — Energy & ESG Analytics Dashboard

**Customer provides (fixed input):** 12 months of energy bills (PDF/Excel) per building · building
list (name, address, type, floor area m², year if known) · EPC if available.

**You deliver (fixed output):** a hosted Power BI dashboard + a 1-page PDF summary — portfolio KPIs
(EUI, kWh, €, CO₂), per-building benchmark + ranking, consumption trend, anomaly flags, and the top
3 quick wins. Every figure tagged *measured / estimated*.

**Your steps (every time, same order):**
1. Intake + check the data against the input checklist.
2. Load into EnergyLens (Tier-1 baseline; bridge the building → B012+ id).
3. **QA every number** against plausible ranges.
4. Export the PDF + share the dashboard link.
5. 20-minute walkthrough call.

**Price:** ~€750–1,200 fixed (reference-phase). **Turnaround:** 3–5 working days. **Revisions:** 1 round.
**Out of scope:** custom visuals beyond the standard set, on-site work, certified certificates, bespoke modelling.

---

## Offer 2 — EPBD / CO₂ Readiness Report  *(headline)*

**Customer provides:** same as Offer 1 + any EPCs.

**You deliver:** a branded PDF report — portfolio carbon (Scope 1/2), **CRREM stranding by building**,
EPBD/GEG readiness per building, ranked retrofit measures with **subsidy-adjusted (BAFA/KfW)** payback,
carbon-cost trajectory, and a prioritised action list. ESRS-E1-aligned; **screening, not certified**.

**Your steps (every time):**
1. Intake + checklist.
2. Load + bridge the building (B012+).
3. Run the compliance + recommendation engines.
4. **QA every figure** (especially payback/savings sanity).
5. Export the report PDF.
6. 30-minute findings call.

**Price:** ~€1,500–3,000 (reference-phase). **Turnaround:** 5–10 working days. **Revisions:** 1 round.
**Out of scope:** certified Energieausweis / iSFP, on-site audit, bespoke modelling, custom report sections.

## Companion — Monitoring subscription

After Offer 1 or 2: a recurring dashboard refresh (monthly or quarterly) on a batch run. ~€99–199/mo.
The proto-SaaS bridge — turns a one-off client into recurring revenue.

---

## Scope guardrails (the anti-fatigue rules)

- **Fixed input → fixed output → fixed price.** A request outside this = a separate quote, never a freebie.
- **Same delivery steps every time** → repeatable, low-fatigue, and you get faster each run.
- **The sample shows exactly what they get** → customers arrive pre-informed, fewer back-and-forths.
- **Founder-delivered (now):** you run the data and export the report. No client login → **no bridge
  automation, no RLS needed** yet. (RLS only when a client pays for live dashboard access.)
- **First paid report = also the real-data test** (messy client data through the pipeline) → QA hard.

---

## How the pieces fit (the clean loop)

Listing (what they get) → **sample PDF** (proof) → client says yes → enter as a **pilot** in `/admin`
→ onboard their building (B012+) → run + QA → **export report PDF** → deliver + findings call → ask
for a **reference** → (optional) monitoring subscription.

Each completed job = income + a real-building reference + one more proof point. Repeat.
