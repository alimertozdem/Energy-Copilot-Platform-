🔴 CRITICAL RULES — READ FIRST
Expand/Collapse Critical Rules
Rule 1: Format Percentage Measures CORRECTLY
All DAX measures ending in "Pct" (e.g., Data Quality Score Pct, Renewable Energy Rate Pct) return values between 0 and 100.
✅ Correct format: 0.0 or 0.00 (NOT "0%" — Power BI multiplies by 100 again!)
❌ Wrong: Format "0%" will display 5000% instead of 50%
Rule 2: All Visuals Have Standard Settings
• Title: OFF
• Background: Transparency 100% (transparent)
• Border: OFF
• Shadow: OFF
This keeps the report clean and lets the accent color system work.
Rule 3: KPI Indicator Setup (6 Fields)
Every KPI Indicator needs exactly these fields configured:
1. Value: Current measure (e.g., Total Consumption kWh)
2. Trend axis: Date field (e.g., Date[Month] or Date[Date])
3. Target goals: PY or fixed target measure
4. Callout text size: 28pt (or 24pt for secondary cards)
5. Callout color: White or page accent color
6. Sparkline: ON, use accent color
Missing any field = blank target or no sparkline.
Rule 4: KPI Indicator Direction
Set in Format pane → Indicator tab → Direction:
• "Low is Good": Consumption, Cost, CO₂, Anomalies
• "High is Good": Savings, Renewable %, Occupancy, Battery SoC
Rule 5: Status Colors (Conditional Formatting)
Use these consistently:
• Good/Success: #00E5A0 (Mint Green)
• Neutral/Warning: #FFC107 (Amber)
• Bad/Critical: #FF4560 (Red)

🎯 KPI INDICATOR SETUP GUIDE (Step-by-Step)
Expand/Collapse Setup Guide

Creating a KPI Indicator Visual

  1. Insert → KPI Indicator visual onto the canvas at specified position
  2. Resize to match spec table (e.g., 210×92 for Card 1)
  3. Data pane (right):
    • Drag current measure to Value well (e.g., Total Consumption kWh)
    • Drag date field to Trend axis (e.g., Date[Month])
    • Drag target/PY measure to Target goals well
  4. Format (paintbrush icon) → Callout:
    • Text size: 28pt (or 24pt for secondary cards)
    • Color: #FFFFFF or page accent color
    • Weight: Bold
  5. Format → Sparkline:
    • Show sparkline: ON
    • Sparkline color: Use page accent (e.g., #00C8FF for Page 1)
  6. Format → Indicator:
    • Direction: "Low is Good" or "High is Good" (see Rule 4 above)
  7. Format → Data labels:
    • Apply format (e.g., #,##0 "kWh" for consumption)
🔧 TROUBLESHOOTING: Target Shows BLANK or Same as Value

Problem: Target goal shows "(Blank)" or same number as current value.

Causes & Fixes:

  • Date table not marked: Go back to Prerequisites step 1. PY measures need this.
  • Only 1 year of data: PY measures can't work with single-year data. Use a fixed target or rolling average instead (e.g., Energy Reduction Target kWh).
  • Wrong measure in Target: Check DAX file — is the PY measure actually defined? Search for "PY" in DAX.
  • Relationship broken: Model view → check relationship Date[Date] → gold_kpi_daily[date] is active (solid line).
