⚠️ CRITICAL FORMAT RULE — APPLIES TO ALL PERCENTAGE CARDS & CHART AXES
All DAX measures that end in Pct already return a 0–100 value (e.g. 25.87 means 25.87 %).
Use the card / axis format string "0.0" or "#,##0.0" — NEVER "0%".
Using "0%" makes Power BI multiply the value by 100 again, producing values like 2587% instead of 25.87 %.
✅ Correct format: "0.0"  |  ❌ Wrong format: "0%"

🔍 DATA VALIDATION RULES — ENFORCE IN SILVER LAYER (prevent wrong values on the report)
R1. Reject rows where energy_kwh < 0 or energy_kwh > 10×P99(building) → physically impossible.
R2. Flag if meter reading gap > 6 hours → set data_quality_flag = 'GAP'. Hide on visuals or mark grey.
R3. Unit must be canonical kWh. Reject Wh / MWh mixed rows → normalise in Silver transform.
R4. CO₂ factor must match (country_code, year) — use EEA annual grid mix factors, NOT a hardcoded value.
R5. EUI only computed when conditioned_area_m² > 0 AND building_status = 'active'. Else → BLANK (Power BI hides).
R6. Occupancy probability must be in [0, 1] (or 0–100 after ×100). Reject <0 or >100.
R7. Forecast rows → hide visual when model_mape_pct > 25 (unreliable). Show banner "Insufficient forecast accuracy".
R8. Certification fields: validate against allowed set → {'LEED', 'BREEAM', 'DGNB', 'HQE', 'WELL', 'ISO 50001', 'EPC A'..'EPC G'}. Reject freetext.
R9. Currency → fixed to EUR. Non-EUR inputs converted at monthly ECB reference rate, date-matched.
R10. Add page-level "Data Freshness" card on every page: [Last Ingestion Timestamp]. If > 48h old → red badge "Data stale".
R11. Invalid-row banner on Portfolio page: if [Invalid Rows Pct] > 2% → show amber banner at top.
🇪🇺 EU REGULATORY ALIGNMENT — THE METRICS & THRESHOLDS USED IN THIS REPORT
EPBD recast (Directive 2024/1275) — in force 28 May 2024, national transposition by 29 May 2026. All new public buildings must be Zero-Emission Buildings (ZEB) by 1 Jan 2028, all other new buildings by 1 Jan 2030. Member States must renovate ≥16% worst-performing non-residential by 2030, ≥26% by 2033. Whole-life carbon reporting mandatory by 2030.
EED (Directive 2023/1791) — EU-level 11.7% final energy reduction by 2030 vs 2020. Member State annual targets.
EPC (Energy Performance Certificate) — harmonised A–G scale per EPBD Art.19. Class A = top-performing, G = worst. Mandatory on sale/rent.
EU Taxonomy — Art.8 + Climate DA — buildings substantially contribute to CCM if: EPC class A, OR top 15% of national stock, OR (for major reno) 30% primary energy reduction.
CSRD / ESRS E1 (Climate Change) — mandatory disclosure of Scope 1, 2, 3 emissions; energy intensity; transition plan alignment with 1.5°C. In force 2024 reporting year (large listed) → 2025+ (all large EU co.).
CRREM (Carbon Risk Real Estate Monitor) — science-based decarbonisation pathway per asset type & country. Example office EU avg target ≈ 52 → <10 kgCO₂e/m²·yr by 2050. Germany office v2 benchmark = 54 kgCO₂/m²·yr (2020). Stranding risk year = first year actual exceeds pathway.
GHG Protocol — Scope 1 (direct, on-site fuel), Scope 2 (purchased electricity/heat — location-based and market-based), Scope 3 (value chain).
ISO 50001 — Energy Management System certification. Not a performance rating, but a governance credential.
LEED (USGBC) · BREEAM (BRE) · DGNB · HQE · WELL — voluntary green-building rating schemes. Tracked in this report as portfolio credentials, NOT as primary performance benchmarks.
